How do I build a website?

  1. Come up with an idea!

    2. Here are some ideas!

    Website Ideas!
    Animals Blog
    Service Video games
    Online class HowTo website
  2. Make a root folder

    3. Name a folder with NO SPACES in a place on your computer. Make sure you remember where this is. If you want images on your website, then put a folder inside that folder called images.

  3. VS code

    4. Make sure you have visual studio code installed on your computer. This is where you will edit your website. Open the folder you created in the last step.

  4. Start editing

    5. Below I will list some of the basic skills you can use when designing a website. For more resources please visit the resources page. Click here for the resources page.

  5. To add a header use the tag h1 through h6. h1 is the biggest header and h6 is the smallest.
  6. To add a paragraph use the tag p.
  7. To link a website use the tag a href="YOUR WEBSITE" target="_blank". This will open your website in a different tab.
  8. To add an image use the tag img src="YOUR IMAGE LINK"
  9. One way we add style is called inline. For example to add a font-color to a header you would use the tag h1 style="color: red;"
  10. One way we add style is called internal. This is when we have the tag style with an id or class inside.
  11. Another way we add style is called external. This is in a seperate folder called style.css and classes and id's. I link this page to my pages on my website to apply them.

5. Now upload your site!

I use something called GitHub, in which I put my folders into the main GitHub one. The domain is pointed to the GitHub server.
